# Contributing to Geopolitical Risk Analysis

Thank you for your interest in contributing! Here's how you can help.

## Getting Started

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/geopolitical-risk-analysis.git`
3. **Create** a branch: `git checkout -b feature/your-feature`
4. **Make** changes and **commit**: `git commit -m "Description"`
5. **Push** to your fork: `git push origin feature/your-feature`
6. **Create** a Pull Request

## Code Style

- Follow PEP 8
- Max line length: 100 characters
- Use type hints
- Document all functions
- Run `black src/` to auto-format

## Testing

- Write tests for new functions
- Run: `pytest tests/ -v`
- Target >80% coverage
- Use descriptive test names

## Documentation

- Update docstrings (Numpy format)
- Update README for major changes
- Add examples where helpful

## Commit Messages

Format: `<type>(<scope>): <description>`

Types:
- `feat`: New feature
- `fix`: Bug fix
- `docs`: Documentation
- `test`: Tests
- `refactor`: Code refactoring

Example: `feat(models): Add HMM clustering`

## Report Issues

- **Bugs**: Include error message and steps to reproduce
- **Features**: Explain motivation and use case
- **Questions**: Use GitHub Discussions

## Questions?

Open a GitHub issue or discussion. We're here to help! 🚀
